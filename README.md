# room-booking-bot-AI
Japanese version of room booking chatbot

**Room Booking Bot** can help you to book, cancel a meeting room and daily meeting room.

[![forthebadge](https://forthebadge.com/images/badges/made-with-python.svg)](https://www.python.org/)[![ForTheBadge built-with-love](http://ForTheBadge.com/images/badges/built-with-love.svg)](https://GitHub.com/Naereen/)

## How to use

1. Run `rasa train` to train model
```console
rasa train
```
2. Run actions
```console
rasa run actions --actions actions
```
3. In a new tab, run `rasa shell` or `rasa x`
```console
rasa shell || rasa x
```
Now, try to chat with Room booking bot.
